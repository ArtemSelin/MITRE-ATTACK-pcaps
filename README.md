# MITRE-ATTACK-pcaps
Данный репозиторий представляет собой открытый банк эталонных pcap файлов в классификации MITRE&amp;ATTACK, предназначенных для тестирования функций безопасности сетевых средств защиты информации, а также иных сетевых устройств, разработки и отладки сигнатур безопасности, исследования логики сетевых атак.
